Detailed Description

Definiciones y prototipos para los algoritmos de ordenamiento, búsqueda y selección.

Este archivo contiene las estructuras de datos principales y las firmas de las funciones utilizadas en el sistema de deportistas basadas en estrategias "Divide y Vencerás".
